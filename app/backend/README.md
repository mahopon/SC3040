# Backend

This is the backend of the SC3040 project.

## ⚙️ Setup

1. Setup python virtual environment

   ```bash
   python -m venv venv

    # Mac/Linux
    source ./venv/bin/activate
    # Windows
    .\venv\Scripts\activate
   ```

1. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

1. Run development server

   ```bash
   python main.py
   ```
